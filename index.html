Interactive Demo using GitHub API Token

Steps to create a personal access token to authenticate with the GitHub API for GitHub users:

  1. Go to GitHub account settings by clicking on your profile photo in the top right and selecting "Settings"
  2. In the left sidebar, click on "Developer settings".
  3. Under "Personal access tokens", click on "Fine-grained tokens" or "Personal access tokens (classic)"
  4. Click "Generate new token" to create a new personal access token.
    5. NOTE: GitHub recommends using fine-grained personal access tokens over the classic personal access tokens, as they provide more security features.
